# Exercise 0101 — Basic Type Hints

## Goal

Learn how to annotate variables using Python's four most common
built-in types:

- `str`
- `int`
- `float`
- `bool`

## Problem Statement

A company is onboarding a new employee.

Several variables already contain valid values, but their type
annotations have been removed.

Replace every `"???"` with the correct type annotation.

Example:

```python
employee_name: "???" = "Alex Rivera"
```

should become:

```python
employee_name: str = "Alex Rivera"
```

## Learning Objectives

After completing this exercise you should be able to:

- recognize common primitive Python types
- annotate variables using `: Type`
- read type annotations in existing code
- identify the difference between strings, integers, floats and booleans
