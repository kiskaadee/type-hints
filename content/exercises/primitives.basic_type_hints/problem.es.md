# Ejercicio 0101 — Type Hints Básicos

## Objetivo

Aprender a anotar variables utilizando los cuatro tipos básicos más
comunes de Python:

- `str`
- `int`
- `float`
- `bool`

## Enunciado

Una empresa está registrando la información de un nuevo empleado.

Las variables ya contienen valores válidos, pero sus anotaciones de
tipo fueron eliminadas.

Reemplaza cada `???` por la anotación correcta.

Ejemplo:

```python
employee_name: ??? = "Alex Rivera"
```

debe convertirse en:

```python
employee_name: str = "Alex Rivera"
```

## Objetivos de Aprendizaje

Al finalizar este ejercicio deberías poder:

- reconocer tipos primitivos comunes en Python
- anotar variables utilizando la sintaxis `: Tipo`
- leer type hints en código existente
- distinguir entre cadenas, enteros, números decimales y booleanos
